# 🎓 Módulo Aulas — acompanhar as aulas

Apoio para as suas **aulas de russo**: registrar o que foi visto, organizar tarefas e tirar dúvidas com o Claude.

## O que tem aqui
| Pasta | Para quê |
|-------|----------|
| `aulas/` | Uma nota por aula (`AAAA-MM-DD-aula.md`) — tema, conteúdo, palavras novas |
| `tarefas/` | Lições de casa e prazos |
| `duvidas/` | Perguntas para levar à aula ou tirar com o Claude |
| `prompts/` | Prompts específicos para aulas (resumir aula, gerar exercícios do tema...) |
| `templates/` | Modelo de nota de aula |

## Fluxo sugerido
1. **Antes da aula:** revise a última nota em `aulas/` e as `duvidas/`.
2. **Durante/depois:** registre a aula em `aulas/` (use o template).
3. **Consolidar:** peça ao Claude para resumir/gerar exercícios (`prompts/`).
4. **Tarefa:** anote em `tarefas/` e marque quando concluir.
5. `Commit-and-sync` no Obsidian.
