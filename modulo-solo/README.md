# 🧍 Módulo Solo — estudo autônomo

Estudo de russo **por conta própria**, com a ajuda do Claude por voz.

## O que tem aqui
| Pasta | Para quê |
|-------|----------|
| `prompts/` | Seus prompts de ensino (tutor, correção, flashcards, gramática...) — **cole os seus aqui** |
| `licoes/` | Uma nota por sessão de estudo (`AAAA-MM-DD.md`) |
| `vocabulario/` | Listas de palavras e campos semânticos |
| `gramatica/` | Anotações de gramática |
| `templates/` | Modelos do Obsidian (ex.: lição diária) |

## Fluxo
1. Escolha um prompt em `prompts/` e cole no app do Claude (voz).
2. Estude / converse / peça correção.
3. Registre em `licoes/AAAA-MM-DD.md` (use o template).
4. `Commit-and-sync` no Obsidian.
