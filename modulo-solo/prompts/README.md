# Prompts

Cada arquivo é um prompt pronto para colar no **app do Claude** (por voz ou texto).

> 💡 Comece toda sessão por [[00-persona-professora]] (ou pela skill `/professora-russo`) para
> "ligar" a professora de russo. Veja [[como-usar]] para a rotina recomendada.

| Arquivo | Para quê |
|---------|----------|
| `00-persona-professora.md` | **Base** — persona de professora de russo para falantes de PT |
| `memoria-25-palavras.md` | Sistema de memória: 25 palavras/tema, com quiz a cada 5 |
| `roleplay-vida-real.md` | Roleplay de situações reais (pedido, entrevista, direções…) |
| `conversa-real.md` | Conversa livre de ~10 min com correção suave |
| `plano-7-dias.md` | Plano semanal completo de estudo (coach de fluência) |
| `como-usar.md` | Rotina e ordem recomendada de uso dos prompts |
| `tutor-conversa.md` | Praticar diálogo em russo com correção suave |
| `correcao.md` | Corrigir frases que você escreveu/falou |
| `flashcards.md` | Gerar flashcards de vocabulário (formato Anki/Obsidian) |
| `gramatica.md` | Explicar um ponto gramatical com exemplos |
| `licao-do-dia.md` | Gerar uma micro-lição completa do dia |

> Mantenha o bloco `---` no topo (frontmatter) para os prompts ficarem organizados e
> pesquisáveis no Obsidian.
