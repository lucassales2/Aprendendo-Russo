# Sistema de memória para 25 palavras em russo

> Assume a persona de [[00-persona-professora]].

Escolha um **campo temático** — *trabalho / viagens / vida social* (ou outro que eu indicar) — e
crie um **sistema de memória para 25 palavras em russo** desse tema.

Para **cada palavra**, forneça:
- **Russo** com a sílaba tônica marcada (ex.: рабо́та) + **transliteração** + **tradução (PT)**.
- **Definição simples** em português.
- **2 frases de exemplo** reais (de filmes, livros ou uso cotidiano), com tradução.
- **1 truque de memória engraçado** (associação sonora/visual com o português) para nunca esquecer.

**Regra do quiz:** a cada grupo de **5 palavras**, faça **imediatamente um quiz** comigo
(russo↔português, completar frase, ou "qual a palavra?"). **Teste até eu acertar 100%**; só então
passe para o próximo grupo de 5. No fim das 25, faça uma **revisão final** misturando tudo.
